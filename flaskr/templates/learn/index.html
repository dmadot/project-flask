Learn

This page explains how to use PlotPie to process and visualize NMR spectra.

Session

Before uploading data, a session must be started. The session creates a temporary workspace where uploaded files are processed and stored during use. This ensures that different users can work independently.

Sessions are temporary and need to be closed after use.

Upload

PlotPie currently supports Bruker NMR datasets.

File Requirements

  • Only Bruker datasets are supported
  • The dataset must be uploaded as a ZIP archive
  • The ZIP file should contain the original Bruker folder structure

The uploaded archive is automatically extracted and processed by the application.

File Naming

Please use simple file names without spaces or special characters, as these may cause issues during processing.

Recommended format:

  • butyl_phenyl_ether.zip
  • sample_1.zip

Avoid names such as:

  • butyl phenyl ether.zip
  • butyl-phenyl-ether (final).zip
  • sample#1.zip

Processing

After uploading, select your file from the list and choose Plot to view the raw FID (Free Induction Decay). Click FFT to apply a Fourier Transform and convert the FID into a frequency-domain spectrum.

Phase Correction

After the FFT, the spectrum may appear distorted. Use the phase correction controls to fix this:

  • P0 — zero-order phase correction. Applies a constant phase shift to the entire spectrum.
  • P1 — first-order phase correction. Applies a phase shift that varies linearly across the spectrum.

Adjust P0 and P1 until the peaks appear symmetric and the baseline is flat. Click Process to apply the changes.

Baseline correction is applied automatically during processing.

Window Size

Use the X-Min / X-Max and Y-Min / Y-Max fields to zoom into a region of interest. Click Process to update the plot.

Peak Detection

PlotPie can automatically detect peaks in the spectrum.

  • Enable Show peaks to mark detected peaks on the plot.
  • Use the Threshold field to set the minimum intensity for peak detection. Enable Show to display the threshold line on the plot.

Adjust the threshold to filter out noise and focus on real peaks.

Integration

Integrals represent the area under a peak and are used to determine the relative number of nuclei.

To add an integral, specify:

  • Name — a label for the integral region.
  • I0 and I1 — the left and right boundaries of the integration region (in ppm).

Enable Show integrals to display them on the plot. Use Reset Integrals to remove all defined integrals.

Export

Once the spectrum is processed, you can save the plot as an SVG file using your browser. Right-click the image and select Save image as.
